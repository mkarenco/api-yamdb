# YaMDb
___

**YaMDb** — API для работы с произведениями искусства (фильмы, книги, музыка и т.п.), отзывами, комментариями и пользовательскими рейтингами.

## 🎯 Цель проекта

**В процессе выполнения этого проекта мы учимся:**

+ работать с чужим кодом и писать код, который будет понятен другим
  
+ следовать единому стилю оформления кода и структуры проекта

+ разрабатывать с учётом технического задания и требований ревью

+ грамотно использовать инструменты контроля версий (Git, GitHub)

+ проводить код-ревью и принимать конструктивную критику

+ применять на практике **Django** и **Django REST Framework** для создания API

Этот проект максимально приближен к реальной работе в команде разработчиков.

## Возможности:

+  Регистрация пользователей

+ Аутентификация по JWT

+ CRUD для произведений, жанров, категорий

+ Оставление отзывов и комментариев

+ Система рейтинга на основе отзывов

+ Роли пользователей: **user / moderator / admin**

## 🛠 Установка и запуск проекта

1️⃣ **Клонируйте репозиторий:**

```bash
git clone https://github.com/api-yamdb.git
cd api-yamdb
```
2️⃣ **Создайте виртуальное окружение и активируйте его:**

```bash
python -m venv venv
source venv/bin/activate  # Linux / Mac
venv\Scripts\activate     # Windows
```
3️⃣ **Установите зависимости:**

```bash
pip install -r requirements.txt
```
4️⃣ **Примените миграции:**

```bash
python manage.py migrate
```
5️⃣ **Создайте суперпользователя (по желанию):**

```bash
python manage.py createsuperuser
```
6️⃣ **Запустите проект:**

```bash
python manage.py runserver
```

## 🔗 Примеры запросов


| Метод | URL                               | Описание                         |
|-------|-----------------------------------|----------------------------------|
| POST  | /api/v1/auth/signup/              | Регистрация пользователя         |
| POST  | /api/v1/auth/token/               | Получение JWT-токена              |
| GET   | /api/v1/titles/                   | Получение списка всех произведений|
| POST  | /api/v1/titles/                   | Добавление произведения             |
| GET   | /api/v1/titles/{id}/              | Детальная информация о произведении |
| GET   | api/v1/titles/{title_id}/reviews/ | Получение списка всех отзывов |
| GET   | api/v1/titles/{title_id}/reviews/{review_id}/| Полуение отзыва по id |
